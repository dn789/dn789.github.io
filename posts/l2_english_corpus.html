This is a tool for automatically generating a corpus of second-language (L2) English from Reddit posts for a variety of native languages (L1s).

You'll need a Reddit account to access the API (authentication instructions here (opens in a new tab)).

The tool is intialized with a dictionary associating L1s to a list of subreddits with speakers of that language (e.g. if I want a corpus of L2 English by German speakers, I'd add subreddits like r/de and r/austria to the config). It uses the Reddit API to get comments from users who post/comment in those subreddits. If a user has a sufficient number of comments in the L1, some of their English comments are added to the corpus for that L1.
